# 01_BinarySearch
자료구조 스터디 1주차 이진탐색(Binary Search) 학습 후, 카카오 겨울 인턴십 이진탐색 문제 풀이

- 이진탐색(이분탐색) 알고리즘 : 리스트에서 탐색 범위를 절반씩 좁혀가며 데이터를 탐색하는 방법
- 탐색 배열 내부의 데이터가 사전에 정렬 되어 있어야만 사용 가능
- 변수 3개 start, end, min를 사용하여 탐색을 진행한다.
